# Medical Marking Services

Система интеграции медицинской клиники с ГИС МТ «Честный Знак» на базе микросервисной архитектуры. Проект разрабатывается в рамках ВКР.

## Описание
Проект предназначен для автоматизации процессов учета маркированных медицинских изделий на базе оперативного складского учета частной медицинской клиники.

### Архитектура проекта:
- **his-adapter**: Адаптер для взаимодействия с МИС клиники.
- **gis-mt-connector**: Модуль интеграции с API «Честного Знака».
- **bff (Backend for Frontend)**: Прослойка для работы пользовательского интерфейса.
- **local-signer-stub**: Эмулятор электронной подписи для тестирования.
